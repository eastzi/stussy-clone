# 1. stussy-clone

>Stussy shoppingMall clone project를 구현함으로써 <br/>
>학습한 내용을 복습하고 새로운 내용들을 습득할 목적으로 해당 PROJECT를 진행하였습니다. <br/>
>처음 진행한 PROJECT라 간단하다고 생각한 기능을 구현함에도 시행착오를 많이 겪고<br/>
>어려움이 많았지만, 힘든 과정 속에서도 많이 배울 수 있어 기억에 남는 PROJECT입니다.
